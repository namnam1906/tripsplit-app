# TripSplit App

Web app สำหรับบันทึกรายรับรายจ่ายตอนเที่ยวกับเพื่อน รองรับ mobile/tablet/desktop

## Features MVP
- สร้างทริป / folder ตามทริป
- บันทึกสมาชิก
- บันทึกรายรับรายจ่าย
- เลือกผู้จ่ายและคนที่ต้องหาร
- คำนวณยอดคืนแบบ Splitwise
- Dashboard
- Upload receipt image เพื่อส่ง OCR ผ่าน backend/n8n
- เตรียมต่อ Google Drive / Google Sheets / Akson OCR / n8n

## Run local
```bash
npm run install:all
npm run dev
```

Client: http://localhost:5173  
Server: http://localhost:8080
